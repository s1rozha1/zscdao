# ZSC DAO - zerosupercycle

Official website for **zerosupercycle DAO** - a Decentralized Autonomous Organization community for everyone passionate about prediction markets and actively trading on Polymarket.

## About

Welcome to zerosupercycle DAO, where we bring together smart members, project founders, coders, and awesome people who love prediction markets. We believe that when we research alpha together, it becomes an enjoyable and rewarding experience for everyone involved.

## Features

- Modern, responsive landing page
- Built with Next.js 14 (App Router)
- TypeScript support
- Optimized images with Next.js Image component
- Animated geometric backgrounds
- Social media integration (X/Twitter, Discord)

## Tech Stack

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** CSS Modules
- **Deployment:** Vercel-ready

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/zscdao.git
cd zscdao
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build
npm start
```

## Deployment

This project is optimized for deployment on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will automatically detect Next.js and configure the build settings
4. Deploy!

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/zscdao)
